AngularJS implementation

When you browse the source code, you'll see the following AngularJS features implemented:

  • modular MVC architecture
  • dependency injection
  • directives
  • filters
  • unit tested, with mocked dependencies
  • 2-way data binding
  • view is updated via changes to the model

Responsive Web Design

The profiles page has been styled responsively, so that the one set of markup displays differently according to viewing device:

  • desktop
  • tablet
  • mobile
